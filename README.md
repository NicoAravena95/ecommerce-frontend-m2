# La Cosecha - Tienda Online

Proyecto de e-commerce para venta de frutas y verduras frescas.

## Sobre el Proyecto

Este es un sitio web de tienda online desarrollado como proyecto del Módulo 2 - Bootcamp Full Stack Python . Vende frutas y verduras frescas con un carrito de compras funcional.

## Tecnologías Usadas

- HTML5
- CSS (Bootstrap 5)
- JavaScript
- LocalStorage para guardar el carrito

## Funcionalidades

- Ver 9 productos en cards
- Agregar productos al carrito
- Ver el carrito en un panel lateral
- Eliminar productos del carrito
- Vaciar todo el carrito
- Página de detalle de cada producto con descripción
- Formulario de contacto
- El carrito se guarda aunque cierres la página

## Cómo Usar

1. Clonar el repositorio:
```bash
git clone https://github.com/NicoAravena95/ecommerce-frontend-m2.git
```

2. Abrir el archivo `index.html` en tu navegador

¡Listo! Ya puedes ver la tienda funcionando.

## Páginas

- **Inicio**: Muestra todos los productos
- **Detalle de Producto**: Info completa de cada producto
- **Contacto**: Formulario para enviar mensajes
